# 🚀 Personal Portfolio Website

A modern, fully responsive, and pixel-perfect personal portfolio website built from scratch. This project showcases my skills in Front-End development, focusing on clean UI/UX, fluid typography, and dynamic features like a fully functional Dark/Light mode.

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_Site-success?style=for-the-badge)](https://abolfazl-web1.github.io/my-portfolio/)

---

## ✨ Key Features

* **Fully Responsive Design:** Optimized for Mobile, Tablet, and Desktop screens using modern CSS (Grid & Flexbox).
* **Dark/Light Mode Toggle:** Implemented with JavaScript and LocalStorage to remember user preferences.
* **Pixel-Perfect UI:** High attention to detail, typography hierarchy, and spacing (White Space management).
* **Smooth Animations:** Subtle hover effects, micro-interactions, and a custom CSS animated Hamburger Menu for mobile navigation.
* **Semantic HTML5:** Built with accessibility and SEO in mind.

---

## 🛠️ Technologies Used

* **HTML5** (Semantic structuring)
* **CSS3** (Custom Properties, Flexbox, CSS Grid, Media Queries)
* **Vanilla JavaScript** (DOM Manipulation, LocalStorage, Event Listeners)
* **FontAwesome** (Icons)
* **Google Fonts** (Raleway)
